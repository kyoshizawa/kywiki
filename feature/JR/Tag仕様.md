# Tags 機能仕様（最終版）

## 基本仕様

| 項目 | 仕様 |
|------|------|
| 形式 | Key-Value 形式（JSON オブジェクト） |
| Key 項目数 | 最大10個 |
| 必須性 | 任意（オプション） |

## Key 制約

| 項目 | 制約 |
|------|------|
| 最大サイズ | 32Byte |
| 文字種 | 半角英数字のみ |
| 先頭文字 | 英字必須（a-z, A-Z） |
| 使用可能記号 | ハイフン（-）、アンダースコア（_） |
| 正規表現 | `^[a-zA-Z][a-zA-Z0-9_-]*$` |

## Value 制約

| 項目 | 制約 |
|------|------|
| 最大サイズ | 128Byte（UTF-8 エンコーディング） |
| 文字計算 | 全角文字は3byte計算 |
| 使用可能文字 | 半角・全角文字 |
| 禁則文字 | ,（カンマ）、"（ダブルクォート）、'（シングルクォート）、:（コロン）、`\r`（復帰文字）、`\n`（改行文字）、`\t`（タブ文字） |
