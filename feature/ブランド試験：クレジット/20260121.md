# 2026/01/21 Terminal API 申請についてのメモ

・決済アプリ "PT-630" でとっているが、アプリ名 "PT-630-TA" など別で申請する。
  試験は同じ内容になる認識。

・Terminal API は有人で申請する

・スコープ： "CVMあり * (接触 | 非接触)" でテスト  

・当面、出荷時の設定も "CVMあり" とする

```
CVMあり = PINレス設定 なし
```

#### ブランド試験実施マトリクス

以下の１２パターン。

〇 : 実施予定

| ブランド | CVMあり * 接触IC | CVMあり * 非接触IC |
|----------|--------------|---------------|
| VISA     | 〇           | 〇            |
| Master   | 〇           | 〇            |
| JCB      | 〇           | 〇            |
| AMEX     | 〇           | 〇            |
| Diners   | 〇           | 〇            |
| discover | 〇           | 〇            |


#### 用語：ファイル拡張子

tpp ファイル : プロジェクト  
tse ファイル : 試験結果を含まないケース  
tsez ファイル :  試験結果を含むケース
